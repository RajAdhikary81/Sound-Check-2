## 📋 Prerequisites / পূর্বশর্ত

Deploy করার আগে এই তথ্যগুলো জোগাড় করুন:

| Variable | Required | কোথা থেকে পাবেন |
|----------|:--------:|------------------|
| `API_ID` | ✅ | [my.telegram.org](https://my.telegram.org) |
| `API_HASH` | ✅ | [my.telegram.org](https://my.telegram.org) |
| `BOT_TOKEN` | ✅ | [@BotFather](https://t.me/BotFather) |
| `STRING_SESSION` | ✅ | `python3 generate_session.py` চালান |
| `MONGO_DB_URI` | ✅ | [MongoDB Atlas](https://www.mongodb.com/atlas) (Free) |
| `OWNER_ID` | ✅ | আপনার Telegram User ID ([@userinfobot](https://t.me/userinfobot)) |
| `LOG_GROUP_ID` | ✅ | Log group ID (বটকে admin করতে হবে) |
| `OWNER_USERNAME` | ❌ | @ ছাড়া আপনার username |
| `SUPPORT_GROUP` | ❌ | সাপোর্ট গ্রুপ লিংক |
| `SUPPORT_CHANNEL` | ❌ | সাপোর্ট চ্যানেল লিংক |
| `YT_COOKIES` | ❌ | YouTube cookies (bot detection bypass) |

git clone https://github.com/RajSukh81/MusicBangla.git

cat > .env << 'EOF'

docker-compose up -d --build

docker-compose logs -f

## 🍪 YouTube Cookies Setup / ইউটিউব কুকি সেটআপ

> YouTube bot detection bypass করতে cookies লাগে। VPS-এ deploy করলে cookies ছাড়াও বেশিরভাগ সময় কাজ করবে।

<details>
<summary><b>কিভাবে Cookies সংগ্রহ করবেন</b></summary>

1. Chrome/Firefox-এ YouTube-এ **login** করুন
2. [Cookie Editor](https://chromewebstore.google.com/detail/cookie-editor/ookdjilphngeeeghgngjabigmpepanpl) extension ইনস্টল করুন
3. youtube.com-এ গিয়ে extension-এ ক্লিক করুন
4. **Export** > **Netscape** ফরম্যাটে export করুন
5. কপি করা content `.env` ফাইলে `YT_COOKIES` এ paste করুন

**অথবা** বটের private chat-এ:
```
/setcookies
```
কমান্ড দিয়ে cookies.txt ফাইল পাঠান।

**Cookie Management Commands:**
```
/cookiestatus   — cookie স্ট্যাটাস চেক
/refreshcookies — cookies রিফ্রেশ
/setcookies     — নতুন cookies সেট (PM only)
```
